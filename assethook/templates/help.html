Help

Assethook is listening for webhooks calls from the Jamf PRO Server (formally known as the JSS). When a webhook is triggered and this application is called, the serial number will be checked against the local database for a name and asset tag. If found, the values will be updated in the Jamf PRO Server.

Requirements:

  • Please see the settings section to configure this application for use with your Jamf PRO Server.
  • You will need an API account that is able to Read and Write to Mobile Device and Computer objects.
  • Create webhooks for device events on your Jamf PRO Server
